HTML

  • An element is a type of content on a webpage.
  • Text elements,header elements, image elements and list elements are the element types.
  • Elements used to divide and describe page : meta related elements, content division elements, semantic elements eg body
  • Head element contains metadata for the browser and elements in this are not visible to the user
  • Body element contains content that is shown to the user and is an example of semantic html element
  • !DOCTYPE is a declaration
  • div creates division in the page and span creates inline division are non semantic elements
  • h1, h2, h3, h4 , h5, h6 are levels of heading and there can only be one h1 per page
  • The alt text in image element is important for screen readers and other assistive technologies
  • Body element has header main and footer
  • section elements helps to organize contents on the page

CSS

  • Cascading Style Sheets (CSS), which is a programming language used by developers to define how webpage content is presented to users
  • Inline css : When a developer wants to style an element directly in the HTML file. Style attribute of elements is used
  • Internal CSS : Embedding entire stylesheet in the header of html and rules apply only to that page but to all the tagged elements . Using style element inside the head element
  • External CSS : Most common and allows all style elements in a separte file. Using link element in the head element.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
